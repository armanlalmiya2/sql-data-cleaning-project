# SQL Data Cleaning Project

This project focuses on cleaning a real-world tech layoffs dataset using SQL.

## Key Tasks Performed
- Removed duplicate records using window functions
- Standardized company, industry, and country names
- Handled null and blank values
- Converted date fields into proper DATE format
- Prepared a clean dataset for analysis

## Tools Used
- MySQL
- SQL (CTEs, window functions, joins)

## Dataset
Tech layoffs dataset containing company, industry, location, and layoff details.
